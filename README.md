# 修仙挂机录 - Vue3 修仙挂机游戏

一款基于 Vue3 + Vite + TailwindCSS 开发的修仙题材挂机游戏，完整实现了修炼、打怪、装备、地图、试炼、转生等核心玩法，适配桌面端与移动端，支持存档导入导出、自动存档。

## 核心功能
1. **修炼系统**：打坐获取修为，修为满后可突破境界，突破需完成对应打怪数量要求
2. **打怪系统**：点击打怪自动攻击直至怪物死亡，随机掉落装备、道具、金币
3. **装备系统**：8大装备部位，8种品质，支持强化、升级、升星，全方位提升属性
4. **地图系统**：多个修仙主题地图，不同地图对应不同怪物与掉落，随境界解锁
5. **商店系统**：可购买强化材料、修为丹、洗点水等道具，支持手动刷新
6. **仙境试炼**：100层挑战玩法，每10层一个BOSS，掉落稀有装备与强化材料
7. **转生系统**：达到最高境界可转生，重置境界但获得全局属性加成，解锁更高上限
8. **属性加点**：每提升一个境界获得属性点，可自由分配到力量、体质、身法、悟性、福缘五大属性
9. **存档系统**：30秒自动存档，支持手动存档、存档导出/导入、游戏重置
10. **全端适配**：针对桌面端与移动端做了响应式优化，修仙主题UI设计

## 境界设定
游戏采用经典修仙境界体系，每个大境界分为10个小层次，突破需满足修为与打怪数量双重要求：
炼气境 → 筑基境 → 金丹境 → 元婴境 → 化神境 → 炼虚境 → 合体境 → 大乘境 → 渡劫境 → 真仙境 → 金仙境 → 太乙金仙境 → 大罗金仙境 → 道祖境

## 安装与运行
### 本地开发运行
1. 安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`
3. 浏览器访问 `http://localhost:3000` 即可进入游戏

### 生产构建
1. 执行构建命令：`npm run build`
2. 构建产物将生成在 `dist` 目录，可直接部署到静态服务器

### Docker部署
1. 构建镜像：`docker build -t xiuxian-game:latest .`
2. 运行容器：`docker run -d -p 8080:80 --name xiuxian-game xiuxian-game:latest`
3. 浏览器访问 `http://localhost:8080` 即可

## 玩法指南
1. **前期发育**：点击打坐获取修为，提升境界，解锁更多功能；点击打怪获取金币与初始装备
2. **属性加点**：优先加悟性提升修为获取速度，或加力量提升打怪效率，根据需求自由分配
3. **装备养成**：打怪掉落装备，穿戴后提升属性；多余装备可分解为材料，用于强化、升级、升星核心装备
4. **地图解锁**：随着境界提升解锁更高等级地图，更高等级地图掉落更好的装备与更多金币
5. **仙境试炼**：拥有一定装备基础后可挑战试炼，通关获得稀有材料与装备，每10层BOSS奖励丰厚
6. **转生突破**：达到渡劫境圆满后可进行转生，获得全局属性加成，大幅提升修炼与打怪效率，解锁更高上限

## 存档说明
- 游戏默认每30秒自动保存进度到浏览器本地存储
- 手动存档：可在存档管理面板点击手动保存，立即同步当前进度
- 存档导出：可导出当前进度为JSON文件，永久保存到本地
- 存档导入：可导入之前导出的JSON存档，恢复游戏进度
- 重置游戏：将清除所有本地存档，恢复到初始状态，操作不可逆
